## 시작하기
tsc app.ts <br>
npm start
## Core Types
1. number ex) 1,5.3,-10
2. string ex) 'Hi', "Hi"
3. boolean ex) true, false
4. object ex) {age: 30}
5. Array ex) [1,2,3]
6. Tuple ex) [1,2] 고정된 크기의 Array (js에 없음)
7. Enum ex) enum{NEW, OLD} 열거형 데이터
8. Any 모든 타입이 가능함 (가능하면 사용안하는게 ts 강점을 살림)
